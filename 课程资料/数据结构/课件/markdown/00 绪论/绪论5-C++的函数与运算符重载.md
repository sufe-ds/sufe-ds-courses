# 函数重载

C++ 允许功能相近的函数在相同的作用域内以相同函数名声明,从而形成重载。方便使用，便于记忆。

例:

```c
int add(int x, int y);  
float add(float x, float y);  
int add(int x, int y);  
int add(int x, int y, int z);
```

# 注意事项

口重载函数的形参必须不同：个数不同或类型不同。

□编译程序将根据实参和形参的类型及个数的最佳匹配来选择调用哪一个

函数。

int add(int x, int y);

int add(int a, int b);

编译器不以形参名来区分

int add(int x, int y);

void add(int x, int y);

编译器不以返回值来区分

# 运算符重载

- 问题：默认运算符不支持自定义类型

- 解决：通过运算符重载让代码更直观

```cpp
class Point {
public:
    int x, y;
    Point(int x, int y): x(x), y(y) {}
};
int main() {
    Point p1(1,2), p2(3,4);
    Point p3 = p1 + p2; // 编译报错！默认不支持Point+Point
    return 0;
}
```

// 重载+运算符后

```txt
Point p3 = p1 + p2; // 合法，等价于p1 operator + (p2)  
cout << p3.x << "", << p3.y; // 输出4,6（符合预期）
```

# 运算符重载

- 运算符重载是 C++ 的多态特性之一，允许为自定义类型（如类、结构体）重新定义运算符的行为。

• 本质：将运算符“包装”成函数，当运算符作用于自定义类型时，调用对应的重载函数。

```cpp
class Point {
public:
    int x, y;
    Point(int x, int y): x(x), y(y) {}
};
int main() {
    Point p1(1,2), p2(3,4);
    Point p3 = p1 + p2; // 编译报错！默认不支持Point+Point
    return 0;
}
```

// 重载+运算符后

```txt
Point p3 = p1 + p2; // 合法，等价于p1 operator + (p2)  
cout << p3.x << "", << p3.y; // 输出4,6（符合预期）
```

include<iostream>

using namespace std;

class Point{

private:

int x, y;

public:

Point(int  $x = 0$  ，int  $y = 0$  )：  $\mathbf{x}(\mathbf{x})$  ，y(y）{}

// 友元声明：operator<<仍需友元（因为它要访问x/y）

friend Ostream& operator<<(ostream& os, const Point& p);

// 全局函数重载<<：合法访问私有成员（因为是友元）

```c
ostream& operator<<(ostream& os, const Point& p) {

os << "(" << p.x << "", "< p.y << ")"; // 友元身份，合法

return os;

}

int main() {

Point p1(1,2), p2(3,4);

//用公有函数getX() getY()获取私有成员，合法！

Point p3(p1.getX() + p2.getX(), p1.getY() + p2.getY());

cout << "p1 = " << p1 << ", p2 = " << p2 << endl;

cout << "p1 + p2 = " << p3 << endl; // 输出：p1 + p2 = (4,6)

return 0;

}

• 须用全局函数重载, 此时 cout << p 会被编译器解释为:

operator<<(cout, p); // 完全符合左操作数是ostream的语法

□若用成员函数方式重载，左操作数必须是当前 Point 类的对象（即 this 指针指向的对象）。因此须写成 p << cout, 不符合习惯
