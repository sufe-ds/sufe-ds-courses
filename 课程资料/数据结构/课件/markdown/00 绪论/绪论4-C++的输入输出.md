# 标准输出

# - cout

□功能：将数据输出到屏幕

□运算符：<<（插入运算符）

□特点：支持链式输出，可输出常量、变量、表达式

□换行与刷新：endl（输出换行并刷新缓冲区）

include<iostream>

using namespace std;

int main() {

int  $a = 25$

// 链式输出

cout << " 整数: " << a

<<", 计算结果: " << 3*5

<< endl; // 换行

return 0;

}

# 标准输入

# • cin

□功能：从键盘读取数据到变量

□运算符：  $\gg$  （提取运算符）

□特点：按空白符（空格/回车 / Tab）分隔数据，自动匹配变量类型

include<iostream> #include<string> using namespace std;

int main() { int age; string name; cout << "请输入年龄和姓名："; cin >> age >> name; // 输入格式：如“18 Tom” cout << "你输入的是：" << name << "", "< age <"岁"; return 0; }

# 文件输入输出

- 定义：对磁盘文件的读写操作

- 依赖头文件：#include <fstream>

- 核心类：专门处理文件流的类（见下表），这三个类为兄弟类，有共同

的基类 basic_fstream, 均在头文件 <fstream> 中定义

<table><tr><td>类名</td><td>功能</td><td>主要操作</td></tr><tr><td>ifstream</td><td>从文件读取数据（输入）</td><td>读（read）</td></tr><tr><td>ofstream</td><td>向文件写入数据（输出）</td><td>写（write）</td></tr><tr><td>fstream</td><td>读写文件（双向操作）</td><td>读 + 写</td></tr></table>

# 文件输入输出操作步骤

- 创建流对象:

□定义ifstream/ofstream/fstream对象

- 打开文件:

□用 open() 函数关联文件（或创建对象时直接指定文件名）

- 读写数据:

□用  $>> / <<$  或成员函数操作

- 关闭文件:

□用 close() 释放资源（对象销毁时自动关闭）
