□函数直接或间接调用自身。

# 函数的参数传递

- 在函数被调用时才分配形参的存储单元

- 实参可以是常量、变量或表达式

- 实参类型必须与形参相符或可隐式转换为形参类型

- 值传递是传递参数值，即单向传递

- 引用传递可以实现双向传递

- 常引用作参数可以保障实参数据的安全

# 值传递和引用传递的区别

# - 值传递:

□函数调用时，复制一份参数的副本传入函数，函数内部操作的是“副本”，与原始变量无关

# - 引用传递:

□函数调用时，不复制副本，而是将原始变量的“引用”（可理解为变量的“别名”或内存地址的快捷方式）传入函数，函数内部操作的是“原始变量本身”

# 例：输入两个整数并交换（值传递）

include<iostream>

using namespace std;

void swap(int a, int b) {

int t = a;

$\mathbf{a} = \mathbf{b}$ ;

$\mathbf{b} = \mathbf{t}$

}

int main() {

int  $x = 5, y = 10$ ;

cout<<"x = "<<x<<" y = "<<y<<endl;

swap(x, y);

cout<<"x = "<<x<<" y = "<<y<<endl;

return 0;

}

# 运行结果：

$\mathrm{x} = 5 \quad \mathrm{y} = 10$

$\mathrm{x} = 5 \quad \mathrm{y} = 10$

# 例：输入两个整数交换后输出（引用传递）

using namespace std;

void swap(int& a, int& b) {

int t = a;

$\mathsf{a} = \mathsf{b}$

$\mathsf{b} = \mathsf{t}$

}

int main() {

int  $x = 5, y = 10$ ;

cout<<"x = "<<x<<" y = "<<y<<endl;

swap(x, y);

}

运行结果：

# 常引用参数传递:

□函数只能读取原始变量的内容，不能修改（编译器会报错阻止修改操作）；

□无需复制参数副本（兼顾值传递的“安全性”和引用传递的“高效性”），是 C++ 中传递大对象（结构体、类、长字符串）的首选方式。

可避免复制带来的内存和时间开销，同时保护原始对象不被修改：

# 例：常引用参数传递

```txt
include<iostream> using namespace std;
```

```javascript
//大结构体（假设包含大量数据）

```

```txt
    int sum = 0;  
    for (int i = 0; i < 100; i++) {  
        sum +=stu.scores[i]; //仅读取 scores，合法  
    }  
    //stu.name = "Tom"; //错误！const 禁止修改结构体成员  
cout << "平均分：" << sum / 100.0 << endl;  
}
```

# 带默认参数值的函数

- 可以预先设置默认的参数值，调用时如给出实参，则采用实参值，否则采用预先设置的默认参数值。

例如:

```c
int add(int x = 5, int y = 6) {
    return x + y;
}  
int main() {
    add(10, 20); //10+20
    add(10); //10+6
    add(); //5+6
}
```

# 默认参数值的说明次序

- 有默认参数的形参必须列在形参列表的最右，即默认参数值的右面不能有无默认值的参数

- 调用时实参与形参的结合次序是从左向右

例:

int add(int x, int y = 5, int z = 6); // 正确  
int add(int x = 1, int y = 5, int z); // 错误  
int add(int x = 1, int y, int z = 6); // 错误

# 默认参数值与函数的调用位置

- 如果一个函数有原型声明，且原型声明在定义之前，则默认参数值必须在函数原型声明中给出；而如果只有函数的定义，或函数定义在前，则默认参数值需在函数定义中给出。

例:

int add(int x = 5, int y = 6);

// 原型声明在前

int main() {

add();

}

int add(int x,int y){

// 此处不能再指定默认值

return  $x + y$ ;

}

int add(int x = 5, int y = 6) {

// 只有定义，没有原型声明

return x + y;

}

int main() {

add();

}

# 函数返回值

- 传值方式返回:

□函数创建返回对象的临时副本，调用者接收复制品，与原始对象无关。局部对象销毁。

- 引用方式返回:

□返回已存在对象的引用（别名），调用者操作引用即操作原始对象。

□需避免悬垂引用，如引用局部变量

# 例：函数返回的传值方式和引用方式

```txt
string get_greeting() { string msg = "Hello, C++"; return msg; // 返回副本 }
```

```cpp
int main() { string greeting  $=$  get_greeting(); cout<< greeting << endl; return 0;   
}
```

```txt
int arr[5] = {1,2,3,4,5};  
int& get_element(int index) {  
    return arr[index]; // 返回引用  
}  
int main() {  
    get_element(2) = 10; // 修改原始元素  
    cout << arr[2] << endl;  
    return 0;  
}
```

# 常引用返回

- 返回已存在对象的只读引用，调用者仅能读取，无法修改原始对象

- 当类的成员函数以“常引用（const T&）”返回时，函数末尾必须加 const 修饰，表示该成员函数是常量成员函数，不修改成员变量。

```cpp
class Person {
public:
    string name;
    const string& get_name() const {
        return name; // 返回常引用
    }
};
int main() {
    Person p = {"Alice", 25};
    const string& name_ref = p.get_name();
    cout << name_ref << endl;
    return 0;
}
```
