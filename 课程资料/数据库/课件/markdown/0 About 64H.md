# About purpose of the course

■ Understand the basic concepts and principles of database  
■ Know how database technology was and will be developed in the future  
Design database application system  
Implement and manage database system

> Lay a foundation for the analysis, design, development and application of large-scale information system software;  
> Lay a foundation for large-scale data storage, manipulation and analysis.
