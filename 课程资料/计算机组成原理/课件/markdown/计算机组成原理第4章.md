# SDRAM

Synchronous DRAM ——同步动态存储器，它的数据带宽为 64 位，接口为 168 Pin DIMM，3.3 V 电压。这是目前使用最广泛普通的一种内存类型。

![](file://~/MinerU/计算机组成原理第4章.pptx-91bafe65-dc30-420e-87a5-4bdb9963dc5c/images/6ea1100e75245761d99e4a2d276b769a33bd38e7ee4633787886a944f41dfc33.jpg)


# DDR SDRAM

DDR (Double Data Rate) DRAM 也可以说是传统 SDRAM 的升级版本，与 SDRAM 一样，DDR 也是与系统总线时钟同步的，不同点在于 DDR 在信号的上升沿与下降沿时都进行数据处理与传输，而 SDRAM 只在信号的上升沿读取数据，因此 DDR 的速度是 SDRAM 的两倍。

![](file://~/MinerU/计算机组成原理第4章.pptx-91bafe65-dc30-420e-87a5-4bdb9963dc5c/images/66d552238418baf2a79ef4bd41346bab778d34912457aa106b0dbf13c1e5993a.jpg)


# RamBus DRAM

无论从性能还是价格来讲，RamBus DRAM 是目前内存世界的王者。RamBus Inc. 宣称这种新的技术能够提供十倍于 DRAM 和三倍于 PC 100 SDRAM 的性能。

- 硬盘转速：是硬盘内电机主轴的旋转速度。硬盘转速以“转/每分钟”来作为单位。RPM值越大，数据传输率就越快，访问时间就越短，硬盘的整体性能也就越好。普通硬盘的转速一般有5400rpm、7200rpm两种，7200rpm高转速硬盘是现在台式机的首选；而对于笔记本用户则是4200rpm、5400rpm为主；服务器对硬盘性能要求最高，常使用的SCSI硬盘转速基本都采用10000rpm，甚至还有15000rpm的

- 接口类型：硬盘接口是硬盘与主机系统间的连接部件，作用是在硬盘缓存和主机内存之间传输数据。不同的硬盘接口决定着硬盘与计算机之间的连接速度，其优劣直接影响着程序运行快慢和系统性能好坏。

IDE 的英文全称为 “Integrated Drive Electronics”,即“电子集成驱动器”。其拥有的价格低廉、兼容性强的特点。其后发展分支出更多类型的硬盘接口,比如 ATA、 Ultra ATA、 DMA、 Ultra DMA 等接口都属于 IDE 硬盘。

SCSI 的英文全称为 “Small Computer System Interface”（小型计算机系统接口），是同 IDE（ATA）完全不同的接口，它一种广泛应用于小型机上的高速数据传输技术。SCSI 接口具有应用范围广、多任务、带宽大、CPU 占用率低，以及热插拔等优点，但较高的价格使得它主要应用于中、高端服务器和高档工作站中。

SATA (Serial ATA) 硬盘又叫串口硬盘, 是未来 PC 机硬盘的趋势。Serial ATA 采用串行连接方式, 具备了更强的纠错能力, 发现错误会自动矫正, 提高了数据传输的可靠性。串行接口还具有结构简单、支持热

![](file://~/MinerU/计算机组成原理第4章.pptx-91bafe65-dc30-420e-87a5-4bdb9963dc5c/images/edd876a05ff30bd118c6f0763aa2b266a2a9167876353afa8a63119fba5397d2.jpg)

半导体材料对一定波长的入射光有足够大的光吸收系数，即要求入射光子的能量大于或等于半导体材料的带隙，使该入射光子能被半导体吸收而激发出光生非平衡的电子空穴对。

PN 结耗尽区的重要作用是分离了两种不同电荷的光生非平衡载流子，在 P 区内积累了非平衡空穴，而在 N 区内积累起非平衡电子。产生了一个与平衡 PN 结内建电场相反的光生电场，于是在 P 区和 N 区间建立了光生电动势（或称光生电压）。

三态门，是指逻辑门的输出除有高、低电平两种状态外，还有第三种状态—高阻状态的门电路。高阻态相当于隔断状态。三态门都有一个 EN 控制使能端，来控制门电路的通断。可以具备这三种状态的器件就叫做三态门。

三态门是一种扩展逻辑功能的输出级，也是一种控制开关。主要是用于总线的连接，因为总线只允许同时只有一个使用者。通常在数据总线上接有多个器件，每个器件通过OE/CE之类的信号选通。如器件没有选通的话它就处于高阻态，相当于没有接在总线上，不影响其它器件的工作。

# Cache 和主存的一致性

• 写直达法（Write-through）

写操作时数据既写入 Cache 又写入主存

写操作时间就是访问主存的时间，读操作时不涉及对主存的写操作，更新策略比较容易实现

• 写回法（Write-back）

写操作时只把数据写入 Cache 而不写入主存

当 Cache 数据被替换出去时才写回主存

写操作时间就是访问 Cache 的时间,

读操作 Cache 失效发生数据替换时,

被替换的块需写回主存，增加了 Cache 的复杂性

# RAID(Redundant Array of Independent Disks) 独立冗余磁盘阵列

1）许多大型机构的数据，如银行的存取款信息、电信的收费记录、税务局的纳税人档案等信息对数据的可靠安全性要求非常高。

![](file://~/MinerU/计算机组成原理第4章.pptx-91bafe65-dc30-420e-87a5-4bdb9963dc5c/images/d0e3c0f483b44cc4f7785de7058689828d4d85a2260eeda734bc4a0d5702c6cc.jpg)


所谓热备份，就是在建立磁盘阵列系统时，将其中一个磁盘指定为后备磁盘，这个磁盘在平常并不操作，当阵列中某一个磁盘发生故障时，磁盘阵列立即让后备磁盘取代故障磁盘，并自动将故障磁盘的数据重建在后备磁盘之上对于要求不停机的大型数据中心中心或控制中心，热备份更是一项重要的功能，因为它可避免晚间或无人守护时发生磁盘故障所引起的种种不便。

![](file://~/MinerU/计算机组成原理第4章.pptx-91bafe65-dc30-420e-87a5-4bdb9963dc5c/images/2362e001970f0cbd2d53a65d61e8e5b815ca4edaa98878df70cef4172571d9b0.jpg)


# RAID 0

把至少两块物理磁盘设备串联一起工作，将数据依次写入到多个物理磁盘中。

优点：在最理想的情况下，硬盘设备的读写性能会提升数倍。

缺点：若任意一块磁盘发生故障将导致整个系统的数据都受到破坏，不具备数据备份和错误修复能力。

# RAID 1

把至少两块物理磁盘设备并联一起工作，将数据同时写入到多个物理磁盘中。

优点：当其中某一块硬盘发生故障后，一般会立即自动以热交换的方式来恢复数据的正常使用。

缺点：多块磁盘设备中写入了相同的数据，因此设备的利用率大幅下降，而且一定程度上增大了系统计算功能的负载。

# RAID 5

parity across disks

![](file://~/MinerU/计算机组成原理第4章.pptx-91bafe65-dc30-420e-87a5-4bdb9963dc5c/images/c1cf1ecdadb3008a5f167dcae66dfce48e7a4310d40263d8d7df83f6a15defdb.jpg)


把磁盘设备的数据奇偶校验信息保存到其他磁盘设备中。RAID5 磁盘阵列组中数据的奇偶校验信息并不是单独保存到某一块磁盘设备中，而是存储到除自身以外的其他每一块应哦按设备上，这样的好处是其中任何一个设备损坏后不至于出现致命缺陷。RAID5 技术实际上没有备份磁盘中的真实数据信息，而是当硬盘设备出现问题后通过奇偶校验信息来尝试重建损坏的数据。RAID5 这样的技术特性“妥协”的兼顾了硬盘设备的读写速度、数据安全性与存储成本问题。

![](file://~/MinerU/计算机组成原理第4章.pptx-91bafe65-dc30-420e-87a5-4bdb9963dc5c/images/85e9aa3f0837f4697e378675f231b3f72117b26d111dd4599758458314b5d1d1.jpg)


![](file://~/MinerU/计算机组成原理第4章.pptx-91bafe65-dc30-420e-87a5-4bdb9963dc5c/images/431cd615f7b5bea598062e5e7ab7e19f42c0bf939fc3f71d6423f39694bd8a46.jpg)


RAID10 即 RAID0+RAID1 的一个组合体，RAID10 技术需要至少 4 块硬盘来组建，其中先分别两两制作成 RAID1 磁盘阵列，以保证数据的安全性；然后再对两个 RAID1 按阵列实施 RAID0 技术，进一步提高硬盘设备的读写速度。这样从理论上讲，只要坏的不是同一组中的所有磁盘，那么最多可以损坏  $50\%$  的硬盘设备而不丢失数据。由于 RAID10 技术继承了 RAID0 的高速写速度和 RAID1 的数据安全性，在不考虑成本的情况下 RAID10 的性能都超过了 RAID5，因此当前成为广泛使用的一种存储技术。

# 不会发生机械故障

# 工作温度范围更大

典型的硬盘驱动器只能在 5 到  $55^{\circ} \mathrm{C}$  范围内工作。而大多数固态硬盘可在  $-10 \sim 70^{\circ} \mathrm{C}$  工作, 一些工业级的固态硬盘还可在  $-40 \sim 85^{\circ} \mathrm{C}$ , 甚至更大的温度范围下工作 (e.g: RunCore 军工级产品温度为  $-55 \sim 135^{\circ} \mathrm{C})$  。

# 体积小重量轻

低容量的固态硬盘比同容量硬盘体积小、重量轻。但这一优势随容量增大而逐渐减弱。直至256GB，固态硬盘仍比相同容量的普通硬盘轻。

# 成本高

每单位容量价格是传统硬盘的  $5 \sim 10$  倍（基于闪存），甚至  $200 \sim 300$  倍（基于 DRAM）。

# 易受外界影响

固态硬盘更易受到某些外界因素的不良影响。如断电（基于DRAM的固态硬盘尤甚）、磁场干扰、静电等。

# 写入寿命有限

写入寿命有限（基于闪存），一般闪存写入寿命为 1000 到 10万次，特制的才能达 100 万到 500 万次。

# 数据难以恢复

传统的磁盘的损坏，通过数据恢复还能挽救一部分数据。但是如果是固态存储，一但芯片发生损坏，要想恢复数据那几乎就是不可能的。使用 RAID 成本非常高。
