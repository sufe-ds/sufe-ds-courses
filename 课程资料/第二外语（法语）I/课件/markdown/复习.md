# 定冠词的形式

le le stylo

I' |'ami

la la valise

I' l'amie

les les stylos, les valises

# 定冠词的用法

1. 表示以前提过的人或事物:

Ex. C'est un film. Dans le film…

2. 表示说话双方熟知的人或众所周知的事物

Ex. Donnez-moi la clé.

3. 虽初次提到, 但受到名词补语（介词 + 名词）限定

Ex. C'est l'appointement d'Dominique.
