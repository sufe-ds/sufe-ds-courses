# Leçon 3 Qui est-ce?

Remarques 语音小知识

- 音节：单词的音节以元音为主体，辅音和元音拼读构成音节，如 la 为一个音节，salut 为 2 个音节

- 开音节和闭音节：以元音音素结尾的为开音节，如 statue；以辅音音素结尾的为闭音节，如 tulipe

- 省音: 少数以元音字母结尾的单词, 后面遇上以元音开始的词, 要省去词末元音字母, 加上省音撇 , 如 c'est = ce + est
